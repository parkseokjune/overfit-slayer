## 정직한 결론

1. **BB는 "횡보장 완충재"가 아니다** — 상승장에서 벌고 위기엔 같이 깨진다 (리뷰 가설 기각).
   BB의 가치 = Sharpe 기여(+0.14)와 상승장 참여 확대이지, 손실 보험이 아님.
2. **진짜 보험은 sma(10,200)** — 단독으론 Sharpe를 깎지만 위기 숏이 포트를 지킨다.
   "Sharpe 희석 = 제거 대상"이 아니라는 교과서적 사례.
3. **2025 whipsaw 개선(프로세스 OOS -0.8%→+5.3%)의 원천은 정적 분산이 아니라 적응(재선택)**.
   → 시스템 정체성 = "보수적 적응형 생존 시스템" (외부 리뷰 방향 재확인).
4. 운용 시사점: 위기 대응은 북 구성이 아니라 sma 보험 + 스탑 + 적응이 담당한다.
   BB 비중 확대는 상승장 베팅 확대를 의미하므로 90일 라이브 후에만 재검토.
